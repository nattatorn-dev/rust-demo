# Rust

## Install

```bash
$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
$ source $HOME/.cargo/env
```

## Initial

```bash
$ cargo init demo-rust
```

## Build (creates binary in target/)

```bash
$ cargo build
```

## Run

```bash
$ cargo run
```

## Test

```bash
$ cargo test
```

## Code Formatting

```bash
$ cargo fmt
```
